# Evidence Packet Addendum

Use when a packet, deck, review, or decision record ignores concrete materials.

```text
Subject: Evidence packet for [record/forum]

Hi [Record Owner],

To keep [record] accurate before [forum/deadline], please attach or link this evidence packet:

- Exact current wording to correct: "[quote]"
- Source proof: [artifact/date/link]
- Supporting metric or decision: [artifact/date/link]
- Owner or contributor field: [name/role]
- Requested replacement or qualification: "[wording]"

Please confirm when this has been posted, forwarded, or appended to [controlling artifact]. If the original thread is not accessible to me, please include me on the confirmation receipt so I know the packet-accessible record has been updated.
```
