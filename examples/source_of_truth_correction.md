# Source-Of-Truth Correction

Use when an old or false record may be treated as final.

```text
Subject: Correcting source-of-truth record for [decision]

Hi [Record Owner],

I want to make sure the current source of truth is accurate before [forum/deadline].

Current record:
- [Exact obsolete or false wording]

Current accurate status:
- Approval status: pending
- True approver: [name/team]
- Open requirement: [requirement]
- Evidence: [artifact/date/link]

Please update [controlling artifact] to say:
"[Corrected wording]. This supersedes the earlier wording '[obsolete wording]' until [true approver] confirms final approval in writing."

If [controlling artifact] is not corrected by [deadline], I will place this neutral addendum in [fallback channel] so the decision record is accurate before [forum].
```
